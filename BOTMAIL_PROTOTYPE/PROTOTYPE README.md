# BotMailv1: Email Auto-Responder & Filter Bot
(This is a lightweight version of BotMail and is a completely open-source Python automation tool that reads incoming emails via IMAP, intelligently classifies them into categories like College, Work, Spam, or General, organises them into folders, and sends automatic replies when appropriate.)

## Features
- Reads unread emails using IMAP
- Classifies emails into College / Work / Spam
- Moves emails into folders
- Sends automatic replies
- 100% open-source
- No Google Cloud, no paid APIs

## Requirements
- Python 3.9+
- Email account with IMAP & SMTP
- App password (recommended)

## Run
python main.py

made with <3
